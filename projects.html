Class Projects

Data Visualization of Economic Performance in Private Sectors

This project utilized Python programming language to evaluate the performance and percent of GDP contribution by private sectors in United States from 2019 to 2020. Python packages such as Tkinter, Pandas and Matplotlib are used to create a simple graphical user interface as well as identify trend patterns on quaterly and yearly basis.

Analysis of Popular Runway Models and Fashion Brands

This team project focused on analyzing data to identify patterns and trends of runway fashion. Our analysis that is conducted primarily on Oracle APEX provide information for people who are interested in runway fashion and models. Final results are displayed in interative reports and charts for clearer understanding.

Airbnb Rental Ratings Analysis

In this project, a data analysis is structured to determine significant variables that affects the rental ratings of Airbnb accommodations. Following a problem solving framework, a regression analysis is conducted on the dataset to achieve actionable results. Microsoft Excel is utilized to clean, analyze and visualize the dataset.
